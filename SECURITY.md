# Security Policy

## Supported Versions

This project provides security updates for the following versions:

| Version | Supported          |
| ------- | ----------------- |
| 5.1.x   | ✅ Yes            |
| 5.0.x   | ❌ No             |
| 4.0.x   | ✅ Yes            |
| < 4.0   | ❌ No             |

If you are using an unsupported version, we recommend upgrading to a supported version to ensure you receive security updates.

---

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Contact**: Email `stellarbot@europe.com or  Stellarbot@usa.com` (or use the GitHub Security Advisory feature).
2. **Information to Provide**:
   - Detailed description of the vulnerability.
   - Steps to reproduce it.
   - Impact assessment (if possible).
   - Any suggested fixes or patches.

3. **Response Time**:
   - We aim to acknowledge all reports within 48 hours.
   - Security fixes will be prioritized in supported releases.

4. **Handling**:
   - Confirmed vulnerabilities will be addressed in the next patch release.
   - If a report is invalid or not reproducible, we will notify the reporter with an explanation.

Please **do not disclose the vulnerability publicly** until a fix has been released.
